Cómo incorporar WhatsApp a nuestro sitio web

¿De qué manera?

  • - Incorporando un botón flotante.
  • - Creando un formulario.

¿Qué necesitamos?

  • Generador de enlace
  • Icono de WhatsApp (Fontawesome)

Botón flotante

  1. Abrir Generador de enlace
    • Ingresar el número de teléfono (*muy importante el código de país)
    • Ingresar el texto predeterminado (*si hacemos click derecho podemos incorporar emojis)
    • Seleccionar "Generar enlace"
    • Seleccionar "Copiar enlace"
  2. Crear la estructura HTML
      <head>
    • <!-- CDN Fontawesome (Para iconos) -->

      • </head>
      <body>
    • <a href=" COPIAR EL LINK GENERADO EN EL PASO 1 " class="float" target="_blank">
        <i class="fab fa-whatsapp"></i>
      </a>
      • </body>

      • </form>
