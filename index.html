¿Qué vamos a ver?

Cómo incorporar WhatsApp a nuestro sitio web

¿De qué manera?

  • - Incorporando un botón flotante.
  • - Creando un formulario.

¿Qué necesitamos?

  • Generador de enlace
  • Icono de WhatsApp (Fontawesome)

Botón flotante

  1. Abrir Generador de enlace
    • Ingresar el número de teléfono (*muy importante el código de país)
    • Ingresar el texto predeterminado (*si hacemos click derecho podemos incorporar emojis)
    • Seleccionar "Generar enlace"
    • Seleccionar "Copiar enlace"
  2. Crear la estructura HTML 
        <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Incorporando WhatsApp</title>

        <!-- Link css local -->
        <link rel="stylesheet" href="css/styles.css">

        <!-- CDN Fontawesome (Para iconos) -->
        <link rel="stylesheet" href="https://pro.fontawesome.com/releases/v5.10.0/css/all.css" integrity="sha384-AYmEC3Yw5cVb3ZcuHtOA93w35dYTsvhLPVnYs9eStHfGJvOvKxVfELGroGkvsg+p" crossorigin="anonymous" />
    </head>
    <body>
        <a href=" COPIAR EL LINK GENERADO EN EL PASO 1 " class="float" target="_blank">
            <i class="fab fa-whatsapp"></i>
        </a>
    </body>
  3. Dar estilos con CSS 
        .float {
        position: fixed;
        bottom: 40px;
        right: 40px;
        width: 60px;
        height: 60px;                       
        background-color: #25d366;
        color: #fff;
        border-radius: 50px;
        font-size: 30px;
        box-shadow: 2px 2px 3px #999;
        z-index: 10;
        display: grid;
        place-items: center;
        text-decoration: none;
    }

    .float:hover {
        color: #25d366;
        background-color: #fff;
    }

Formulario de contacto

  1. Crear un formulario en HTML (*muy importante que cada input tenga su id) 
        <form onsubmit="enviarWhatsapp()">
        <input type="text" id="name" placeholder="Nombre">
        <input type="number" id="phone" placeholder="Teléfono">
        <textarea placeholder="Daja tu comentario" id="message"></textarea>
        <input type="submit" name="submit" value="Contactar por whatsapp" />
    </form>
  2. Crear archvio de JavaScript y vincularlo 
        <body>

        <!-- Incorporar el link del archvio js al final del body -->
        <script src="main.js"></script>
    </body>
  3. Crear código de JavaScript 
        function enviarWhatsapp() {
        const name = document.getElementById("name").value;
        const phone = document.getElementById("phone").value;
        const message = document.getElementById("message").value;
        const url = "https://api.whatsapp.com/send?phone=+ NÚMERO DE WHATSAPP &text=" +
                "Hola" + "%0a%0a" +
                "Mi nombre es: " + name + "%0a" +
                "Teléfono: " + phone + "%0a" +
                message;
        window.open(url, '_blank').focus();
    }
